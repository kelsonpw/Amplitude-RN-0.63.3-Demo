# Amplitude-RN-0.63.3-Demo

The purpose of this repository is to demo a React Native application on v0.63.3 using @amplitude/react-native@2.0.11 without errors on any platform.

**Dependencies**:
```json
"dependencies": {
  "@amplitude/react-native": "2.0.11",
  "react": "16.13.1",
  "react-native": "0.63.3"
}
```
